# Site Web : Thé Bulle

Projet scolaire réalisé dans le cadre du cours Web 2 au programme de Techniques d‘intégration multimédia du Collège Montmorency. 2026.

**Projet réalisé par :** Vincent Quesnel

## Contenu du projet
- Intégration complète de la maquette (Hero, Équipe, Contact, Footer).
- Fichier `DOCUMENTATION.md` contenant les justifications techniques.
- Dossier `/public/images/` incluant tous les assets (SVG, PNG, icônes).

## Installation
Ouvrir le fichier `index.html` dans un navigateur moderne pour visualiser l'intégration.
